### Passos iniciais

Clone esse projeto para sua maquina:

```bash
git clone url_do_projeto
```

Para inciar pacotes essenciais do projeto desse repositorio execute:

```bash
npm install
```

Para Executar o projeto desse repositorio execute:

```bash
npm run dev
```
